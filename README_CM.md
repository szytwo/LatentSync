## 安装

```
conda create --prefix ./venv python==3.10.13

conda activate ./venv

pip install -r ./api_requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

docker build -t latentsync1.5:latest .  #构建镜像

docker-compose up -d #后台运行容器

```
